# robot-marketing

ربات تلگرام بازاریابی. با یک دستور روی هر لینوکسی نصب می‌شود — بدون git clone، بدون توکن. اسکریپت قبل از هر مرحله ازت می‌پرسد (y/n).

## نصب (یک دستور، کپی کن)

```bash
curl -fsSL https://raw.githubusercontent.com/mariamtchelidze66/robot-marketing/main/install.sh | bash
```

یا با wget:

```bash
wget -qO- https://raw.githubusercontent.com/mariamtchelidze66/robot-marketing/main/install.sh | bash
```

بعد از نصب، فایل `.env` را از روی `.env.example` بساز و کلیدها را داخلش بگذار.
